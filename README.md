# hypercube
Minecraft clone in haskell focussed on performance.

## downloading
### with git
```shell
$ git clone https://github.com/noughtmare/hypercube.git
$ cd hypercube
```

## building
```shell
$ stack build
```

## running
```shell
$ stack exec hypercube
```
